Several files of documentation are available according to your usage :

  3. User documentation : Users Guide

  4. General informations about components : Components
    ... a very macroscopic documentation on available components.

  5. General architecture : Developpers Guide
    ... documentation on the general architecture and conception guidelines.

  6. For developpers : ImplementersGuide
    ... rules and helps for components implementation.

  7. EOS gui (module EOS_IHM) : GUIGuide
    ... describes how to develop and how to use the EOS_IHM module.
